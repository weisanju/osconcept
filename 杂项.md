# lamba表达式的情形

1. 组成 

   参数列表 ->  代码块

2. 情形

   1. 无参 无返回

      ```java
      @FunctionInterface
      public interface InterfaceWithNoParm{
          void run();
      }
      InterfaceWithNoParm a= () -> sysout(1)
      ```

   2. 有参无返回

      ```java
      void run(String s)
      InterfaceWithNoParm = (String s) -> sysout(a)
      InterfaceWithNoParm = s -> sysout(s)
      ```

   3. 无参有返回

      ```java
      String run();
      InterfaceWithNoParm = () -> "helloWorld"
      ```

   4. 有参有返回

      ```
      String run(String s);
      InterfaceWithNoParm = (s) -> s
      ```
